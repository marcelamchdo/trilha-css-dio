# Desafio 01: Criando sua primeira Landing Page com HTML e CSS

O objetivo do desafio era construir uma Landing Page com HTML e CSS, colocando em prática os fundamentos do CSS, as propriedades básicas da linguagem de estilização, além de trabalhar com as unidades de medidas relativas e absolutas que aprendemos ao longo da trilha.
